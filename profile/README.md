### Breach72 — PDPA Incident Response Command Center

[![Live Demo](https://img.shields.io/badge/demo-pdpa.404sleepnotfound.com-22d3ee?style=flat-square)](https://pdpa.404sleepnotfound.com)
[![Stack](https://img.shields.io/badge/Next.js_16-React_19-CF_Workers-D1-Drizzle-Tailwind_4-6366f1?style=flat-square)](#)

**PDPA Hackathon Track 4: Incident Response and Breach** — ระบบรับมือเหตุละเมิดข้อมูลส่วนบุคคลภายในกรอบ 72 ชั่วโมง สำหรับ DPO, Incident Commander และ Security Team ออกแบบให้เป็นกลางต่อประเภทหน่วยงาน ใช้ได้กับหน่วยงานรัฐ องค์กรปกครองส่วนท้องถิ่น รัฐวิสาหกิจ สถาบันการศึกษา และภาคเอกชน

| Capability · ความสามารถ | Detail · รายละเอียด |
|---|---|
| Command Center | นาฬิกานับถอยหลัง 72 ชั่วโมงจากเวลาที่องค์กรทราบเหตุ |
| Decision Tree | Severity × Likelihood + checklist PDPA 8 ข้อ + จุดตัดสินใจของ Controller |
| Playbook and Runbook | แบ่งเฟส 0–4, 4–24, 24–48, 48–72 ชั่วโมง และหลัง 72 ชั่วโมง |
| Communication Templates | ร่างรายงานสำหรับหน่วยงานกำกับ เจ้าของข้อมูล Controller/Processor และผู้บริหาร |
| Guardrail | ห้าม export เมื่อข้อมูลไม่ครบหรือยังไม่มีคำตัดสินใจจากผู้มีอำนาจ |
| Audit Trail | บันทึกทุกการกระทำ — เปิดเหตุ, ประเมิน, อนุมัติ, override, export |

### ChatUp — แชท + กระเป๋าเงินดิจิทัล

[![Goal](https://img.shields.io/badge/goal-chatup.404sleepnotfound.com-f59e0b?style=flat-square)](#)
[![Stack](https://img.shields.io/badge/React_19-CF_Workers-D1-KV-R2-Durable_Objects-PWA-8b5cf6?style=flat-square)](#)

**WeChat เวอร์ชันไทย** — แอปแชท + กระเป๋าเงินดิจิทัลเพื่อ Financial Inclusion กลุ่มเป้าหมายคือคนที่ไม่มีบัญชีธนาคาร สมัครด้วยเบอร์มือถือ — แชท โอนเงินในแชท สแกนจ่าย QR อั่งเปา จ่ายบิล ครบในแอปเดียว UI ตัวอักษรใหญ่ รองรับเสียงอ่าน ภาษาไทยเป็นหลัก + EN/เมียนมา/ลาว/เขมร

| Capability · ความสามารถ | Detail · รายละเอียด |
|---|---|
| Chat · แชท | ข้อความส่วนตัว, กลุ่ม, แชร์รูป, ส่งไฟล์ — UI ใหญ่ รองรับเสียงอ่าน |
| Wallet · กระเป๋าเงิน | โอนเงินในแชท, สแกน QR จ่าย, อั่งเปา, จ่ายบิล — ครบในแอปเดียว |
| 5 Languages · 5 ภาษา | ไทย (หลัก) + EN, เมียนมา, ลาว, เขมร |
| Easy Signup · สมัครง่าย | ใช้แค่เบอร์มือถือ — ไม่ต้องมีบัญชีธนาคาร |
| Monorepo | pnpm workspaces: `apps/web` (PWA), `apps/api` (Worker), `packages/shared` |
